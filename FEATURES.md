# Yes Duty Free - 서비스 기능 안내

> 현재 제공 중인 기능과 추가 예정 기능을 정리한 문서입니다.

---

## 📌 서비스 개요

**Yes Duty Free**는 면세 상품을 조회·장바구니 담기·주문까지 할 수 있는 웹 서비스입니다.  
Supabase 기반 회원/상품/주문/장바구니/배송지/찜하기가 연동되어 있으며, 회원 등급에 따른 적립률·배송비가 적용됩니다.

---

## ✅ 현재 제공 중인 기능

### 1. 메인·쇼핑

| 기능 | 설명 |
|------|------|
| **홈** | 히어로 배너, 카테고리 바로가기(면세 에디션, 글로벌 트렌드, 라이브 등), 실시간 인기 면세템 목록 |
| **전체 카테고리** | 카테고리 그리드 진입 (클릭 시 해당 카테고리 상품 목록으로 이동) |
| **카테고리별 상품 목록** | 카테고리 선택 시 해당 상품만 필터링하여 표시 |
| **상품 상세** | 상품 이미지, 브랜드/카테고리, 가격·할인율, 옵션 선택, 수량 선택, 장바구니 담기·즉시 구매하기 |
| **즉시 구매하기** | 해당 상품을 장바구니에 넣은 뒤 장바구니 페이지로 이동 (장바구니에서 삭제 후 결제 가능) |
| **특가 / 베스트** | 특가(deals), 베스트(best) 상품 목록 페이지 |

### 2. 회원

| 기능 | 설명 |
|------|------|
| **로그인** | 이메일·비밀번호 로그인 (Supabase Auth 연동) |
| **회원가입** | 이메일·비밀번호·이름·전화번호·기본 배송지(국내/해외, 다음 우편번호 API) 입력 후 가입 |
| **로그아웃** | 세션 종료 |

### 3. 장바구니

| 기능 | 설명 |
|------|------|
| **장바구니 담기** | 상품 상세에서 수량·옵션 선택 후 담기 (로그인 필요) |
| **장바구니 페이지** | 담긴 상품 목록, 수량 변경, **삭제**(삭제 시 "정말 삭제하시겠습니까?" 확인 모달) |
| **주문하기** | 장바구니에서 주문서(결제) 페이지로 이동 |

### 4. 배송지

| 기능 | 설명 |
|------|------|
| **배송지 관리** | 마이페이지에서 진입, 국내(다음 우편번호 API)·해외(Address Line) 주소 등록/수정/삭제, 기본 배송지 설정 |

### 5. 주문·결제

| 기능 | 설명 |
|------|------|
| **주문서(결제) 페이지** | 배송지 선택, 결제 수단, **적립금 사용**(전액 사용 가능), 총 결제 금액 확인 후 주문 생성 |
| **주문 완료** | 주문 번호 표시, 마이페이지/쇼핑 계속하기 링크 |
| **적립금 사용** | 결제 시 보유 포인트 범위 내에서 차감, `orders.used_points`·`users.points` 반영 |

### 6. 회원 등급·포인트

| 기능 | 설명 |
|------|------|
| **등급** | Basic / Premium / VIP (월 구매액 기준 자동 갱신) |
| **적립률·배송비** | Basic 1%·유료배송, Premium 2%·무료배송, VIP 3%·무료배송 등 (규칙: `lib/constants/membership.ts`) |
| **적립** | 주문 완료 후 등급에 맞춰 포인트 적립, `users.points` 갱신 |
| **등급별 혜택 모달** | 마이페이지에서 "등급별 혜택 보기"로 안내 |

### 7. 마이페이지

| 기능 | 설명 |
|------|------|
| **프로필** | 이름·등급·적립금 잔액 표시 |
| **찜한 상품** | 찜한 상품 그리드, 클릭 시 상품 상세 이동, 찜 해제 시 "정말 삭제하시겠습니까?" 확인 모달 |
| **주문 내역** | 주문 번호·날짜·상태·총액 리스트 |
| **주문 상세** | 행 클릭 시 모달로 주문 상세(상품·금액·배송지 등) 표시 |
| **배송지 관리** | 배송지 관리 페이지로 이동 |

### 8. 찜하기

| 기능 | 설명 |
|------|------|
| **찜하기/해제** | 상품 카드·상품 상세·마이페이지 찜 목록에서 하트 버튼으로 토글 (로그인 시) |
| **찜 해제 확인** | 찜 해제 시 "정말 삭제하시겠습니까?" 확인 모달 |

### 9. 라이브

| 기능 | 설명 |
|------|------|
| **라이브 목록** | 라이브 섹션·전체 화면 스타일 뷰 (현재 목업 데이터, 실시간 스트리밍 미연동) |

### 10. 기타 UI

| 기능 | 설명 |
|------|------|
| **검색창** | 헤더 검색 오버레이, 인기 검색어 표시 (실제 검색 로직·결과 페이지 미구현) |
| **관리자** | 관리자 로그인·패널 UI (주문/배송 등 관리 기능은 기본 수준) |

## 🔜 추가 예정 기능

### 우선 진행 예정

| 기능 | 설명 |
|------|------|
| **리뷰** | 상품별 리뷰 작성·조회 (DB 설계 있음, API·UI 연동 예정) |
| **쿠폰** | 주문 시 쿠폰 선택·적용, 할인 금액 반영 (`coupons`, `user_coupons` 테이블 활용) |
| **검색** | 상품명/브랜드 검색 API·검색 결과 페이지, (선택) 자동완성 |

| 기능 | 설명 |
|------|------|
| **배송 추적** | 주문 상세에 송장번호(`tracking_number`), 택배사 링크 표시 |
| **관리자 강화** | 상품 등록/수정, 주문·회원 조회·상태 변경 등 |
| **재고 표시** | 품절/재고 부족 시 장바구니 담기·구매 버튼 비활성화 등 |

| 기능 | 설명 |
|------|------|
| **포인트 내역** | 적립/사용 내역 조회 페이지 |
| **실시간 알림** | Supabase Realtime 기반 알림 |
| **환율 표시** | `exchange_rates` 테이블 활용 환율 표시 |
| **관세 계산기** | 국가별 관세율 계산 |
| **쿠폰 자동 발급·1:1 문의** | 조건별 쿠폰 발급, 문의 채널 |
| **배너·이벤트 관리** | 배너/이벤트 CRUD 및 노출 |
| **실제 결제 PG** | Toss Payments 등 PG 연동 (현재는 주문 생성만) |
