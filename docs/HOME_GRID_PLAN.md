# 홈 그리드(바로가기) 기획

> 상단 8개 아이콘 클릭 시 이동 경로 정리

---

## 현재 상태

| 아이콘 | 라벨 | 현재 이동 | 화면 |
|--------|------|-----------|------|
| 🍳 | 면세 에디션 | `trend_edition` | **빈 화면** |
| 📈 | 글로벌 트렌드 | `trend_global` | **빈 화면** |
| ✈️ | 프리주문 | `trend_pre` | **빈 화면** |
| 🛍️ | 로컬마켓 | `category` (푸드) | ✅ 카테고리 푸드 목록 |
| 📺 | 라방 모아보기 | `live` | ✅ 라이브 페이지 |
| ✨ | 메이커스 | `all_categories` | ✅ 전체 카테고리 |
| 🎫 | 멤버십 쿠폰 | `deals` | ✅ 특가 상품 목록 |
| ⌛ | 한정특가 | `trend_deal` | **빈 화면** |

---

## 기획 제안 (즉시 적용)

**빈 화면이 나오지 않도록** 기존 페이지로 매핑.

| 아이콘 | 라벨 | 제안 이동 | 설명 |
|--------|------|-----------|------|
| 🍳 | 면세 에디션 | **베스트(best)** | 면세 인기/에디션 상품 = 베스트 셀러와 동일한 목록 |
| 📈 | 글로벌 트렌드 | **베스트(best)** | 글로벌 인기 상품 = 베스트 셀러 |
| ✈️ | 프리주문 | **전체 카테고리(all_categories)** | 프리주문 전용 페이지 없음 → 카테고리 둘러보기로 유도 |
| 🛍️ | 로컬마켓 | `category` (푸드) | 유지 |
| 📺 | 라방 모아보기 | `live` | 유지 |
| ✨ | 메이커스 | `all_categories` | 유지 |
| 🎫 | 멤버십 쿠폰 | `deals` | 유지 |
| ⌛ | 한정특가 | **특가(deals)** | 한정특가 = 특가 상품 목록과 동일 |

---

## 추후 확장 (선택)

- **면세 에디션** · **글로벌 트렌드**: DB에 `tag` 또는 `collection` 추가 후, 전용 목록 API·페이지로 분리
- **프리주문**: 예약/프리오더 전용 테이블·상품 타입 추가 후 전용 페이지 구현
- **한정특가**: `deals` 페이지 내에서 “한정특가” 필터/탭 추가

---

**적용**: 위 제안대로 홈 그리드 클릭 시 `best` / `all_categories` / `deals`로만 이동하도록 수정함.
