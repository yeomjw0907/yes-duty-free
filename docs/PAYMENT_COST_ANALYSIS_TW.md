# 대만 서비스 결제 수수료 비용 분석 (Payment Cost Analysis – Taiwan)

> Yes Duty Free 대만 시장 서비스 시, **PortOne 통합** vs **단일 PG** 수수료 시뮬레이션  
> 작성일: 2026-02-10

---

## ✅ 결제 연동 계획 (확정)

- **PG 연동**: **Stripe**로 진행.
- 대만(TWD) 및 해외 카드 결제 대응, 수수료·구현 단순성 기준으로 Stripe 단일 연동 방안 채택.

---

## 1. 전제 조건

- **서비스 지역**: 대만 (TWD 결제 수요 가정)
- **비교 옵션**
  - **A안**: PortOne 위에서 통합 + 크로스보더(대만) + 하위 PG
  - **B안**: Stripe 단일 연동 (대만/해외 카드)
  - **C안**: 대만 현지 PG 단일 연동 (ECPay / NewebPay)

---

## 2. 수수료 구조 요약

### 2.1 PortOne (한국 기반 통합 PG)

| 구분 | 내용 | 비고 |
|------|------|------|
| **결제 연동(OPI)** | 월 **순거래액** 구간별 고정료 | KRW 기준 |
| | 5천만 원 미만: **무료** | |
| | 5천만 원 ~ 1억 원 미만: **10만 원/월** | |
| | 1억 원 ~ 5억 원 미만: **20만 원/월** | |
| **크로스보더** (해외 결제) | 거래액의 **0.5%** | **최소 USD 500/월** |
| **PG사별 수수료** | PortOne 아래 연동하는 PG사 수수료는 **별도** | 한국 PG 또는 현지 PG |

- 대만은 PortOne 문서상 **Payment Channels – Taiwan** 지원으로 크로스보더 대상 국가에 포함됨.
- 즉, **PortOne(0.5% + OPI) + 대만 쪽 실제 결제 처리 수수료**가 동시에 발생하는 구조로 보면 됨.

### 2.2 Stripe (대만/해외)

| 구분 | 내용 |
|------|------|
| 카드 결제 | **2.9% + 30¢** (건당) – 일반적 표준 요율 |
| 다중 통화/해외 | 환전 시 1~3% 등 추가 가능 (통화·지역에 따라 상이) |
| 고정 월비 | 없음 (종량) |

### 2.3 대만 현지 PG (참고)

| PG | 카드 수수료(참고) | 비고 |
|----|-------------------|------|
| **ECPay (綠界)** | 약 2% ~ 2.75% | 최저 $0.16 USD 등 건당 하한 있을 수 있음 |
| **NewebPay (藍新)** | 약 2.5% 수준 | 정산 3~5 영업일 |

- 위 비율은 일반 가맹점 기준이며, 실제는 거래량·계약에 따라 달라질 수 있음.

## 3. 시나리오 정의

- **결제 통화**: TWD (대만 달러) 또는 USD/KRW 정산 중 하나로 가정.
- **월 거래액**: 소규모 / 중규모 / 대규모 세 구간으로 시뮬레이션.
- **건당 평균 결제액**: TWD 3,000 (약 USD 95, KRW 12만 원 수준)으로 가정.

| 시나리오 | 월 거래액 (TWD) | 월 거래액 (USD, 약) | 월 결제 건수 (대략) |
|----------|------------------|----------------------|----------------------|
| 소규모 | 300,000 | ~9,500 | ~100건 |
| 중규모 | 1,500,000 | ~47,500 | ~500건 |
| 대규모 | 6,000,000 | ~190,000 | ~2,000건 |

- 환율: TWD 1 ≈ 0.0315 USD, 1 USD ≈ 1,260 KRW 가정 (시뮬 시점 기준 반영 가능).

---

## 4. 시뮬레이션 (월간 비용)

### 4.1 A안: PortOne 통합 (크로스보더 대만 + 하위 PG)

**가정**
- PortOne **크로스보더** 사용: 거래액의 **0.5%**, 최소 **USD 500/월**.
- 하위 PG는 대만 카드 결제 수수료만 단순화: **2.5%** (NewebPay/ECPay 중간값).
- OPI: 순거래액을 **KRW**로 환산해서 적용 (TWD→USD→KRW).

**순거래액 KRW 환산 (대략)**
- 소규모: 9,500 USD × 1,260 = **약 1,197만 KRW**
- 중규모: 47,500 USD × 1,260 = **약 5,985만 KRW**
- 대규모: 190,000 USD × 1,260 = **약 2억 3,940만 KRW**

**OPI 적용**
- 5천만 미만: 무료  
- 5천만 ~ 1억: 10만 원/월  
- 1억 ~ 5억: 20만 원/월  

| 시나리오 | 월 거래액 (USD) | PortOne 크로스보더 (0.5%) | 크로스보더 최소 (USD 500) | OPI (KRW) | 하위 PG (2.5%) | **총 수수료 (USD, 대략)** |
|----------|------------------|----------------------------|----------------------------|-----------|----------------|----------------------------|
| 소규모 | 9,500 | 47.5 | **500** | 0 | 237.5 | **500 + 237.5 ≈ 737.5** |
| 중규모 | 47,500 | 237.5 | 500 → 237.5 적용 | 100,000 KRW ≈ 79 | 1,187.5 | **237.5 + 79 + 1,187.5 ≈ 1,504** |
| 대규모 | 190,000 | 950 | 950 | 200,000 KRW ≈ 159 | 4,750 | **950 + 159 + 4,750 ≈ 5,859** |

**총 수수료율 (거래액 대비)**  
- 소규모: 737.5 / 9,500 ≈ **7.76%** (최소 500 USD 때문에 매우 높게 나옴)  
- 중규모: 1,504 / 47,500 ≈ **3.17%**  
- 대규모: 5,859 / 190,000 ≈ **3.08%**  

→ **대만만 단일 시장이고 거래량이 작을 때는 PortOne 크로스보더 최소 USD 500이 부담**이 됨.

---

### 4.2 B안: Stripe 단일 (대만/해외 카드)

- **2.9% + 30¢/건** 적용.
- 30¢ = 약 TWD 9.5, USD 0.30.

| 시나리오 | 월 거래액 (USD) | 건수 | 2.9% | 30¢×건수 | **총 수수료 (USD)** | **수수료율** |
|----------|------------------|------|------|----------|----------------------|--------------|
| 소규모 | 9,500 | 100 | 275.5 | 30 | **305.5** | **3.22%** |
| 중규모 | 47,500 | 500 | 1,377.5 | 150 | **1,527.5** | **3.22%** |
| 대규모 | 190,000 | 2,000 | 5,510 | 600 | **6,110** | **3.22%** |

- 해외 카드/다통화 사용 시 1~3% 추가될 수 있어, 표에는 포함하지 않았음 (추가 가정 시 별도 계산 권장).

---

### 4.3 C안: 대만 현지 PG 단일 (ECPay/NewebPay)

- 카드 **2.5%**만 적용 (건당 하한 무시하고 단순화).

| 시나리오 | 월 거래액 (USD) | 2.5% | **총 수수료 (USD)** | **수수료율** |
|----------|------------------|------|----------------------|--------------|
| 소규모 | 9,500 | 237.5 | **237.5** | **2.5%** |
| 중규모 | 47,500 | 1,187.5 | **1,187.5** | **2.5%** |
| 대규모 | 190,000 | 4,750 | **4,750** | **2.5%** |

- 현지 법인/현지 계약 가능 여부에 따라 실거래 수수료는 2%~2.75% 등으로 조정해 보시면 됨.

## 6. 대만 서비스 시 결론 및 권장

1. **거래량이 작을 때 (월 약 1만 USD 미만)**  
   - PortOne 크로스보더 **최소 USD 500/월** 때문에 **A안이 가장 비쌈**.  
   - **Stripe(B안)** 또는 **대만 현지 PG(C안)**이 훨씬 유리.

2. **중·대규모로 커졌을 때**  
   - A안은 수수료율이 약 3.08~3.17%로, Stripe(3.22%)와 비슷해짐.  
   - 다만 A안은 **PortOne OPI + 크로스보더 + 하위 PG**가 겹치므로, **순수 수수료만 보면 C안(현지 PG 단일)이 가장 저렴**.

3. **PortOne을 쓸 만한 경우**  
   - **한국(KRW) + 대만(TWD)** 등 **여러 국가/통화**를 한 플랫폼에서 묶고 싶을 때.  
   - 이미 한국 PG는 PortOne으로 쓰고 있고, 대만은 크로스보더로 추가하는 구조가 합리적일 때.  
   - **대만만** 단일 시장이고 현지 법인/대리인이 있으면 **C안(ECPay/NewebPay 등)** 단일이 비용상 유리.

4. **실무 체크**  
   - PortOne 크로스보더 **대만 채널**의 실제 수수료·정산 통화(TWD/USD/KRW)는 **onboarding@portone.io** 등으로 한 번 더 확인.  
   - 대만 현지 PG는 **정산 주기, 최소 거래액, 계약 조건**도 같이 비교하는 것이 좋음.
